# Data Directory

JSON storage for Country Farm Matugga. Auto-created by PHP.

Files:
- website-settings.json — contact info, about text
- products.json — product names, prices, descriptions  
- quotes.json — quote requests from public site
- content.json — hero, stats, images, about paragraphs, FAQs, testimonials
- media.json — uploaded file metadata

Do NOT delete this folder or its .htaccess.
